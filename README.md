# 💖 爱心表白动画网页

一个浪漫的网页动画项目，通过粒子动画形成心形图案，配合温馨的文字和背景音乐，完美适合向心爱的人表达心意。

## ✨ 特性

- 🎨 **渐变色粒子动画** - 220个渐变色文字粒子从四周飞入
- 💝 **心形聚合效果** - 粒子沿心形参数方程精确定位，形成完美爱心
- 🎵 **背景音乐循环** - 支持自定义背景音乐，营造浪漫氛围
- 🔄 **自动循环播放** - 动画完成后2秒自动重新开始
- 📱 **响应式设计** - 适配各种屏幕尺寸
- 🎭 **多阶段动画** - 粒子扩散 → 回收聚合 → 心形定型 → 脉搏跳动 → 爆炸散开

## 🎬 动画流程

1. **初始阶段**：粒子从屏幕四周向外扩散
2. **回收阶段**：粒子回收并向中心聚拢
3. **成型阶段**：粒子沿心形曲线精确定位
4. **文字淡出**：文字逐渐消失，显示心瓣形状
5. **心跳效果**：爱心进行一次呼吸式跳动（2.2秒）
6. **爆炸散开**：粒子向四周飞散并淡出
7. **等待重启**：2秒后自动重新播放

## 🚀 使用方法

### 1. 准备文件

确保以下文件已准备好：
- `heart.html` - 主网页文件
- `audio/miss.mp3` - 背景音乐文件
- `icon/heart.png` - 网页图标文件

### 2. 自定义内容

在 `heart.html` 中找到以下部分进行个性化修改：

#### 修改标题
```html
<title>All my true love for LXQ</title>
```

#### 修改中心文字
```html
<div class="center-title" id="title">FOR YOU ♥ 小雪天</div>
```

#### 自定义表白文字
在 JavaScript 的 `tips` 数组中修改或添加你想要的文字：
```javascript
const tips=[
  '想念在风里飘','月亮为你亮','喜欢你很久了',
  '你是我的心动','温柔落在人间',
  // ... 添加更多文字
];
```

#### 更换背景音乐
替换 `audio/miss.mp3` 文件，或修改音频路径：
```html
<audio id="Audio" src="audio/your-music.mp3"></audio>
```

### 3. 打开网页

直接用浏览器打开 `heart.html` 文件即可观看效果。

> 💡 **提示**：部分浏览器可能需要用户交互才能自动播放音乐，首次打开时点击页面任意位置即可启动音乐。

## ⚙️ 参数调整

### 调整粒子数量
```javascript
const TOTAL = 220; // 默认220个，可调整为 200~280
```

### 调整爱心大小
```javascript
const s = 16; // 默认16，可调整为 12~22
```

### 调整动画时长
```javascript
const TOTAL_TIME_MS = 35000; // 总动画时长（毫秒）
```

### 调整循环等待时间
```javascript
setTimeout(play, 2000); // 2000ms = 2秒，可自定义
```

### 调整粒子动画速度
```javascript
el.style.setProperty('--dur', (rand(6.8,8.6))+'s'); // 6.8~8.6秒
```

## 🌟 技术亮点

- **心形参数方程**：使用数学参数方程精确绘制心形曲线
- **CSS动画优化**：使用 `will-change` 和硬件加速提升性能
- **响应式字体**：使用 `clamp()` 函数适配不同屏幕
- **渐变背景**：双层径向渐变营造柔和背景
- **模糊效果**：爆炸阶段添加动态模糊增强视觉效果
